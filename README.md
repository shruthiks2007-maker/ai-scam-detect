# AI Scam Detector 🚨

## 📌 Project Overview
AI-powered system that detects scam and fraudulent messages.

## ❗ Problem Statement
Many users receive scam messages and cannot easily identify whether they are safe or harmful.

## 💡 Solution
This project analyzes messages using AI-based logic and classifies them as:
- ✅ Safe
- ⚠️ Suspicious
- ❌ Scam

## ⚙️ Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- GitHub

## 🚀 Features
- Real-time scam detection
- Simple and user-friendly interface
- Fast response
